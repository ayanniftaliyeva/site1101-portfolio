## Logic Gates Simulation
### A Deep Dive into Boolean Algebra & Binary Logic

I led the development of an interactive simulation that demystifies foundational Boolean logic concepts. This project showcases my ability to bridge technical complexity with intuitive user interface design.

**What I Did:**
- Architected the core Boolean logic engine (AND, OR, NOT gates)
- Designed a clean, accessible UI that helps students experiment and learn
- Implemented input validation and error handling
- Created interactive visualizations for logic gate behavior

**Technical Skills:** Logic design, user testing, validation systems

## Hour of Code (HOC) Event
### Strengthening Algorithmic Thinking & Digital Literacy

I coordinated and facilitated an Hour of Code event focused on teaching algorithmic thinking to students from outside the city center.

**What I Did:**
- Organized event logistics and volunteer coordination
- Implemented core knowledge from our Information Systems course to build the project's technical and instructional framework
- Facilitated hands-on coding sessions and provided mentorship
- Documented the event through photography and videography

**Impact:** Managed 23 students from suburban/rural areas (merkez olmayan) using core Information Systems principles to strengthen algorithmic thinking
